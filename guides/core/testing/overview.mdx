We provide fake network providers for you to use when testing your integration. 
These providers mimic real-world providers and support static placeholders to influence the behavior of the API and test various flows. 
Operations on these providers will not incur any charges.

## Test Providers

The following test providers are currently available:

| Test provider | Description |
| --- | --- |
| test | General-purpose test provider supporting a wide range of test flows. |
| t11 | Test provider mimicking the behavior of provider p11. |
| t12 | Test provider mimicking the behavior of provider p12. |

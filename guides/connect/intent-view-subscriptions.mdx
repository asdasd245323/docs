---
title: "Intent: viewSubscriptions"
description: "How do I show all client subscriptions to a user?"
---

Will open Connect on the subscriptions overview page.

![viewSubscriptions.jpg](https://i.gigscdn.net/docs/v1/connect-session-viewSubscriptions.jpg)

### Creating a Connect Session with viewSubscriptions intent

```bash
curl --request "POST" \
  --url "/api/projects/${GIGS_PROJECT}/connectSessions" \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${GIGS_TOKEN}" \
  --data '{
    "callbackUrl": "https://example.com",
    "intent": {
      "type": "viewSubscriptions"
    },
    "user": "usr_0U2ViuFW0Z7P8Y2S1L76mfADMT6m"
  }'
```

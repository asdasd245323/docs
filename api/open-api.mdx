This page provides a comprehensive table with links to Gigs OpenAPI specifications, along with concise descriptions of their functionality and use cases.

| Name | Description |
| --- | --- |
| [Gigs Core API](/api/latest/core) | API to manage voice & data plans for smartphones, wearables and IoT in 200+ countries. |
| [Gigs Payments API](/api/latest/payments) | Manage orders, payments, and payment methods within Gigs. |
| [Gigs Connect API](/api/latest/connect) | Manage sessions for Connect. |
| [Gigs Billing API](/api/latest/billing) | Manage invoices and credit notes within Gigs. |
| [Gigs Events](/api/latest/events) | Events can describe actions, resource state changes, or specific conditions having been met. |
| [Gigs Reminders](/api/latest/reminders) | Reminders are automated, time-based notifications to identify when users are stuck and need to take action. |

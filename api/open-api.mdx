This page provides a comprehensive table with links to Gigs OpenAPI specifications, along with concise descriptions of their functionality and use cases.

| Name | Description |
| --- | --- |
| [Gigs Core API](/api/gig-core-api/plans/list) | API to manage voice & data plans for smartphones, wearables and IoT in 200+ countries. |
| [Gigs Payments API](/api/gigs-payments/schemas/payment) | Manage orders, payments, and payment methods within Gigs. |
| [Gigs Connect API](/api/gigs-connect/connect-sessions/create-a-connect-session) | Manage sessions for Connect. |
| [Gigs Billing API](/api/gigs-billing/invoices/list-all-invoices) | Manage invoices and credit notes within Gigs. |
| [Gigs Events](/api/gigs-events/schemas/com.gigs.subscription.activated) | Events can describe actions, resource state changes, or specific conditions having been met. |
| [Gigs Reminders](/api/gigs-reminders/schemas/com.gigs.reminder.porting.requested) | Reminders are automated, time-based notifications to identify when users are stuck and need to take action. |

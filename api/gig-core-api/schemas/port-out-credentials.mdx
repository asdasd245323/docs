---
title: "PortOutCredentials"
description: "Port out credentials issued by network provider and used for the porting process at the target provider."
---

<ResponseField name="object" type="string">
  Type of object is always `portOutCredentials`.

  Allowed values: `portOutCredentials`
</ResponseField>

<ResponseField name="accountNumber" type="string | null">
  The account number on the service provider. Account number is `null` if no account number is issued in the respective country.

  Example: `"123456789"`
</ResponseField>

<ResponseField name="accountPin" type="string">
  The account pin from the service provider.

  Example: `"1234"`
</ResponseField>

<ResponseField name="portOut" type="string">
  Unique identifier of the port out this credentials are for.

  Example: `"pto_0SNlurA049MEWV4DWEaT4SaeDI8s"`
</ResponseField>

<RequestExample>
```json Example
{
  "object": "portOutCredentials",
  "accountNumber": "123456789",
  "accountPin": "1234",
  "portOut": "pto_0SNlurA049MEWV4DWEaT4SaeDI8s"
}
```
</RequestExample>
